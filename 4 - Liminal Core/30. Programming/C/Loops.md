##### General:
They are literally JavaScript.

---> `while`:

`while ( i < 5) {`
	`printf("%d\n", i);`
	`i++;`
`}`

---> `do...while`:

`do {`
	`printf("%d\n", i);`
	`i++;`
`} while ( i < 5 );`

---> `for`:

`for ( int i = 0; i < 10; i++) {`
	`printf("%d\n", i);`
`}`
