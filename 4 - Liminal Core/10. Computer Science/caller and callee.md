##### Base Principle:
Both are functions, the caller is the request for a function (the callee) to be executed

##### Example:

int main()
{
	sum(3, 4);        <--- Caller
}

int sum( int a, int b)         <--- Callee
{
	int summe = a +b;
	return summe;
}
